# Análisis de Ventas 2024 · SupermercadoAR 🛒

Dashboard de análisis de ventas de una cadena de supermercados argentina,
construido con SQL Server, Power Query y DAX en Power BI.

## Herramientas
SQL Server · Power Query · DAX · Power BI

## Contenido
- Base de datos relacional con modelo estrella (4 tablas)
- Consultas SQL con window functions (LAG, particiones)
- Transformaciones en Power Query
- Métricas DAX: YTD, MoM, margen, facturación ajustada por inflación
- Dashboard interactivo con filtro por provincia

## Principales hallazgos
- Diciembre fue el mes de mayor facturación con el mayor salto mensual
- Buenos Aires concentra la mayor facturación por volumen poblacional
- Carnes y Bebidas son las categorías más vendidas
- La inflación 2024 distorsiona fuertemente la comparación nominal entre meses
